# 🔥 ProgrEVER — Авто-прогревер Telegram аккаунтов

Бот для автоматического прогрева Telegram аккаунтов по двум стратегиям.

## Статусы аккаунтов

| Эмодзи | Статус |
|--------|--------|
| 🟢 | Идеально прогрет (score ≥ 85%) |
| 🟡 | Хорошо прогрет, нужно ещё (score ≥ 55%) |
| 🔴 | Плохо прогрет (score < 55%) |
| ⚫ | Нет сессии |
| ⚪ | Новый аккаунт (день 0) |
| 🟣 | Трастовый донор (пишет другим аккаунтам) |

## Стратегии прогрева

### Мануал 1 (Приоритетный — Трастовый)
1. **День 0** — Холд 24 часа, аккаунт отлёживается
2. **День 1** — Смена описания профиля
3. **Дни 2–5** — Трастовые (🟣) аккаунты пишут на прогреваемый; вступление в каналы (2/день)
4. **День 6+** — Аккаунт готов писать первым (до 8 человек в день), начинает со стикера

### Мануал 2 (Автономный)
- Вступление в 5–10 каналов в день
- Взаимодействие со @SpamBot (кнопки)
- Визит к @BotFather
- DM между прогреваемыми аккаунтами
- Создание групповых чатов с названиями типа «Огородники», «Одногруппники»
- Обмен стикерами и фото
- Ротация аватарок (установка на день 5, удаление на день 10)
- Постепенное повышение активности (2 недели)

## Установка

### 1. Клонировать / распаковать проект

```bash
cd progrever
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить .env

Скопируйте `.env.example` в `.env` и заполните:

```env
BOT_TOKEN=токен_от_BotFather
API_ID=ваш_api_id
API_HASH=ваш_api_hash
ADMIN_IDS=ваш_telegram_id
```

Получить `API_ID` и `API_HASH`: https://my.telegram.org/apps

### 4. Создать сессии для аккаунтов

Для каждого аккаунта запустите:

```bash
python session_login.py +79001234567
```

Файл `.session` сохранится в папку `sessions/`.

### 5. Запустить бота

```bash
python main.py
```

## Структура файлов

```
progrever/
├── main.py              # Точка входа
├── config.py            # Настройки
├── database.py          # База данных SQLite
├── warming_engine.py    # Планировщик прогрева
├── session_login.py     # Утилита авторизации
├── strategies/
│   ├── manual1.py       # Трастовая стратегия
│   └── manual2.py       # Автономная стратегия
├── bot/
│   ├── handlers.py      # Обработчики команд
│   ├── keyboards.py     # Клавиатуры
│   └── states.py        # Состояния диалогов
├── sessions/            # .session файлы аккаунтов
├── media/               # Фото для аватарок
├── .env                 # Ваши настройки (не коммитить!)
└── requirements.txt
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/status` | Быстрый статус всех аккаунтов |
| `/addaccount` | Добавить новый аккаунт |
| `/cancel` | Отменить текущее действие |

## Расписание

Цикл прогрева запускается автоматически:
- **10:00 МСК** — утренний цикл
- **18:00 МСК** — вечерний цикл

Принудительный запуск через кнопку «🔥 Запустить прогрев сейчас».

## Добавление аватарок

Поместите фото (.jpg/.png) в папку `media/` — бот будет автоматически устанавливать и менять их.

## Важно

- Не используйте слишком много действий в сутки — Telegram банит за спам
- Для Мануала 1 нужен хотя бы 1 трастовый (🟣) аккаунт
- Сессии хранятся локально в папке `sessions/` — берегите их
